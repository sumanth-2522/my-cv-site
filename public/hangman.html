Play Hangman

About the game

To play this game, you need to guess an everyday phrase such as a hot potato for example. You need to hit the keys to start guessing and for each guess a ' _ ' will be replaced if that guess is part of the phrase. If not, you will get 6 chances to guess before the game is Over.

Rules

Please fellow below rules to play this game

  • Please Enter your name in the textbox below and click on submit button
  • After submitting your name please click on START to start the game
  • After Clicking on start, you can start pressing keys to guess the Phrase
  • Please enter Alphabets only
  • If you want to play again, make sure you enter your name again and hit submit
  • There is a hidden hint available on the page to help you complete the game
  • You will get 6 tries to guess the phrase. Good luck!
