Think of a number

Play Think of a number

In this game, a number between 1 and 100 is randomly generated by the computer

You must guess which number was generated. There will be hints such as when you are too close to guessing, the colour of the text box turns red, if you are too far away from the number, the text box turns blue.

Rules

Please fellow below guideines to play this game

  1. User must only enter numbers between 1 - 100
  2. Do not refresh while playing the game
  3. Do not hit back on the browser while playing the game
  4. Click GUESS to start the game
  5. Click RE-START to restart the game
